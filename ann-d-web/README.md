# ANN-D Health Advisor (Static UI)

Standalone UI built with **HTML + CSS + JavaScript** (no frameworks).

## Run

- Open `ann-d-web/index.html` directly in a browser, or
- Run a local server (recommended):
  - PowerShell: `cd ann-d-web; python -m http.server 5500`
  - Open: `http://127.0.0.1:5500`

## Notes

- Microphone button is UI-only (no speech-to-text).
- “Speak” uses the browser Speech Synthesis API.
- Educational guidance only (not a medical diagnosis).
